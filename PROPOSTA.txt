	Primeiramente deve-se criar uma classe JogoCostumizado que herda a classe Jogo, a qual deve haver como atributo um arraylist, que deve ser organizado pelo construtor e definidas as pontuações do dominó. O método criaPeca deve fazer repetições do tamanho da lista e deve chamar o construtor de Peca com os elementos invocados pelos indices do arraylist como parâmetro. Além disso a classe rodada deve agir de forma tradicional, sem tipos de vitórias e declarado empate quando os dois passam a vez, visto que uma maior variedade de peças pode dar problema com o método atualizado para o Dominó Butal 2.0.
	Na classe Peca deve haver a troca de nomes dos atributos e métodos e o tipo para string, caso ocorra de alguma peça personalizada não usar números. No construtor deve haver uma condição de somente existir soma do lado esquerdo e direito se os parâmetros puderem ser lidos como inteiros. Além disso o método getSomaMao da classe Jogador deve retornar 0 caso o atributo seja nulo.
